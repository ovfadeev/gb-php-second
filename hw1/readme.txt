4. Придумать наследников класса из п.1. Чем они будут отличаться?
5. Дан код:
class A {
    public function foo() {
        static $x = 0;
        echo ++$x;
    }
}
$a1 = new A();
$a2 = new A();
$a1->foo();
$a2->foo();
$a1->foo();
$a2->foo();
Что он выведет на каждом шаге? Почему?
1234 - функция foo не является статической в классе, и статическая переменная сохраняет свои значения

Немного изменим п.5
class A {
    public function foo() {
        static $x = 0;
        echo ++$x;
    }
}
class B extends A {
}
$a1 = new A();
$b1 = new B();
$a1->foo();
$b1->foo();
$a1->foo();
$b1->foo();

1122 - Т.к. объявляется класс B то статическая переменная тоже обнуляет своё значение для этого класса
